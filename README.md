puppet-kiosk
===================
Puppet module to install kiosk browser

The module has been tested on
-------------
- Ubuntu Server 12.04.4 LTS

Limitations
-------------
This module has been built on and tested against Puppet 3.2.3 and higher.

Packages
-------------
- Xorg
- Openbox
- Squid3
- Midori

Dependencies
-------------
- none so far
